# Desafio 01: Criando primeira Landing Page com HTML e CSS

Desafio 01: Criando sua primeira Landing Page com HTML e CSS
Desafio prático desenvolvido durante a Trilha de CSS da DIO (Digital Innovation One). O objetivo principal foi construir uma Landing Page do zero, aplicando os conceitos fundamentais de estruturação web e estilização moderna.

🚀 Sobre o Projeto

O projeto consiste na criação de uma página de captura (Landing Page) responsiva e visualmente atraente. A atividade foi essencial para entender como os elementos se comportam na tela e como o CSS transforma uma estrutura simples de HTML em uma interface de usuário profissional.

🛠️ Tecnologias Utilizadas

HTML5: Estruturação semântica da página.

CSS3: Estilização, layout e design responsivo.

🧠 O que foi colocado em prática?

Este desafio cobriu conceitos cruciais para qualquer desenvolvedor Front-End:

Fundamentos do CSS: Sintaxe, seletores (id, classe, tag) e aplicação de estilos.

Propriedades de Layout: Alinhamento de elementos, espaçamentos (margin e padding) e posicionamento.

Unidades de Medida: Entendimento prático e diferenciação entre unidades absolutas (como px) e unidades relativas (como em, rem e %), garantindo melhor flexibilidade ao layout.

Estilização de Elementos: Customização de tipografia, cores de fundo, bordas e imagens.
